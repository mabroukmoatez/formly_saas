# Dashboard Qualité - Design Specifications

## Page Overview
- **File**: Dashboard Qualité
- **Dimensions**: 1728px × 1184px
- **Layout Type**: Sidebar + Main Content + Right Sidebar

## Color Palette
| Usage | Color | Hex |
|-------|-------|-----|
| Primary | Blue | #007AFF |
| Secondary | Orange | #FF7700 |
| Success | Teal | #26C9B6 |

## Typography
- **Primary Font**: Poppins (Headings, UI)
- **Secondary Font**: Inter (Body text)
- **Brand Font**: Urbanist (Logo, Navigation)

## Key Components
1. **Système Qualité Card**
   - Donut chart with 3 segments
   - Legend: Procédure (Orange), Modèle (Teal), Preuve (Yellow-green)

2. **Indicateurs Qualiopi**
   - 32 circular indicator buttons
   - Active state: Teal border
   - Inactive state: Gray border

3. **Actions & Taches**
   - Task cards with category badges
   - Priority indicators
   - Due dates
